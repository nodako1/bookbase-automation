# scene_13

## このシーンの固定役割

重要ポイント③の導入。3つ目の視点・方法・実践ポイントを、一目で分かる形にする。

scene_13は、scene_05やscene_09と同じく重要ポイントの入口である。ただし3つ目のポイントは、動画全体の流れの中で「解決策」「実践」「仕上げ」に近い役割になることが多いため、視聴者に「ここから最後の重要ポイントに入る」と分かる画像にする。

## scene_13で毎回抽出する可変データ

```json
{
  "scene": 13,
  "fixed_role": "重要ポイント③の導入",
  "point_3_label": "重要ポイント③を短く表す言葉",
  "point_3_core_message": "scene_13で一番伝えるべき内容",
  "point_3_type": "practice / solution / mindset / framework / habit / decision_rule / final_perspective のいずれか",
  "visual_metaphor": "重要ポイント③に合う視覚的な比喩",
  "exact_text_elements": [
    "重要ポイント③",
    "短いラベル",
    "短い補足メッセージ"
  ]
}
```

## point_3_type の選び方

- `practice`：具体的に実践する内容を紹介する場合。
- `solution`：問題に対する解決策を紹介する場合。
- `mindset`：考え方や姿勢を紹介する場合。
- `framework`：型や整理法を紹介する場合。
- `habit`：習慣化や継続に関する内容の場合。
- `decision_rule`：判断基準や選び方を紹介する場合。
- `final_perspective`：動画全体の最後の視点や、まとめに近い重要ポイントの場合。

## 画像内テキストルール

- 画像内テキストは3つ以内にする。
- 固定してよいテキストは「重要ポイント③」のみ。
- その他の「短いラベル」「短い補足メッセージ」は毎回の原稿から生成する。
- 原稿本文のコピペ、20文字以上の長い説明、4つ以上のテキスト要素、英語テキストは禁止。

## 推奨構図

- 構図A：第3ポイントカード型。中央または左側に「重要ポイント③」のカード、右側に現在のpoint_3_typeに合った象徴モチーフを置く。
- 構図B：最後の扉・最終ステップ型。扉、3枚目のカード、最終ステップ、光の道などで3つ目のポイントに入る感じを表現する。
- 構図C：実践への橋渡し型。point_3_type が `practice` / `solution` の場合に使用し、理論から行動へ移るイメージを橋・矢印・道・ツールカードで表現する。
- 構図D：整理された型の提示型。point_3_type が `framework` / `decision_rule` の場合に使用し、カード、枠、整理された図で型が見えるようにする。

## 避けること

- チェックリストを固定しない。
- plannerを固定しない。
- 手元のノート構図を固定しない。
- scene_12のコメントCTAと似た手元構図にしない。
- scene_18の実践シーンと混ざる構図にしない。
- ただの作業中画像にしない。
- 長文や英語テキストを入れない。
- 今回の本の重要ポイント③の具体語句を恒久テンプレートに固定しない。
